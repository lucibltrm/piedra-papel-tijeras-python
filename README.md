# 🎮 Piedra, Papel o Tijeras

Proyecto del **Curso de Data Analyst** - Digital House

## 📝 Descripción

Juego clásico de Piedra, Papel o Tijeras para **dos jugadores** desarrollado en Python.  
Permite ingresar los nombres de los jugadores, validar las jugadas y ofrece la opción de jugar múltiples rondas.

## ✨ Características

- Validación de jugadas (solo acepta "piedra", "papel" o "tijeras")
- Insensible a mayúsculas y minúsculas
- Opción de jugar de nuevo al finalizar cada partida
- Manejo de errores en las entradas del usuario

## 🛠 Se usó

- Python 3
- Estructuras condicionales (`if`, `elif`, `else`)
- Bucles (`while`)
- Manejo de strings (`.strip()` y `.lower()`)

## 🚀 Cómo ejecutarlo

1. Copia este repositorio o descarga los archivos.
2. Abrí una terminal en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

```bash
python piedra_papel_tijeras.py
